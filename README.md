# Implementação Trigonométrica

Projeto desenvolvido em C com o objetivo de implementar funções trigonométricas sem utilizar a biblioteca matemática padrão da linguagem.

## Métodos Implementados

### Conversão de Ângulos

* `deg_to_rad(double n)` - Converte graus para radianos.
* `rad_to_deg(double n)` - Converte radianos para graus.

### Funções Trigonométricas

* `m_sin(double rad)` - Calcula o seno de um ângulo em radianos.
* `m_cos(double rad)` - Calcula o cosseno de um ângulo em radianos.
* `m_tan(double rad)` - Calcula a tangente de um ângulo em radianos.

* `m_sec(double rad)` - Secante.
* `m_csc(double rad)` - Cossecante.
* `m_cot(double rad)` - Cotangente.
  
## Métodos Planejados

### Funções Trigonométricas Inversas

* `m_asin(double x)` - Arco seno.
* `m_acos(double x)` - Arco cosseno.
* `m_atan(double x)` - Arco tangente.

### Funções Hiperbólicas

* `m_sinh(double x)` - Seno hiperbólico.
* `m_cosh(double x)` - Cosseno hiperbólico.
* `m_tanh(double x)` - Tangente hiperbólica.

## Tecnologias

* Linguagem C
* CMake
* GCC

## Objetivo

Este projeto foi desenvolvido para estudar trigonometria, séries de Taylor, aproximações numéricas e implementação de bibliotecas matemáticas.
